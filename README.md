# GTNH-monitor-backend
The FastAPI-based backend of GTNH monitor

## Run the Server

```bash
uvicorn main:app --reload
```

## Environment Build

```bash
pip install "fastapi[all]"
```

- Check the [tutorial of FastAPI](https://fastapi.tiangolo.com/tutorial/)
